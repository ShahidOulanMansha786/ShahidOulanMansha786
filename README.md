## What I have built

### [Repairo](https://github.com/ShahidOulanMansha786/repairo-backend) — Car Repair Marketplace
A production-grade marketplace where car owners post repair leads and nearby shops submit quotes in real time.

- Designed **normalized database schema** (10+ tables) with Flyway migrations on PostgreSQL
- Built **geospatial lead matching** using PostGIS — shops only see leads within their service radius
- Implemented **escrow payment system** with automatic release logic and scheduled jobs
- Real-time quote delivery via **WebSocket/STOMP** — no polling, no delays
- **Firebase Firestore** chat between car owners and shops + FCM push notifications
- OTP-based auth with **JWT + rotating refresh tokens** and Redis rate limiting
- Deployed on **AWS using Docker**

> `Java 17` · `Spring Boot 3` · `PostgreSQL` · `PostGIS` · `Redis` · `Firebase` · `WebSocket` · `AWS S3` · `Kotlin` · `Jetpack Compose` · `Next.js` · `TypeScript`
